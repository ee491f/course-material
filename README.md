# EE491F Data Driven Web Development

Getting Started
---------------
1. install docker
2. install docker-compose
3. clone repository: `git clone --recursive https://github.com/zhaol/ee491f.git`
4. run bootstrap.sh: `./bootstrap.sh`
5. start service: `docker-compose up`
